# 分支的使用

## 创建和切换分支
创建
```
git branch dev
```
切换
```
git checkout dev
```
创建并切换
```
git checkout -b dev
```
**切换分支后看到的工作区和当前分支一致**

## 查看当前分支
打星号的是当前分支
```
git branch
```

## 合并到master分支
```
git merge dev
```

## 删除分支
```
git branch -d dev
```

## 解决分支冲突
合并步骤出现"Conflict"字样，可以使用以下语句查看冲突的文件。
```
git status
```
此后修改文件，Git用`<<<<<<<`，`=======`，`>>>>>>>`标记出不同分支的内容。
最后将分支删除即可。
